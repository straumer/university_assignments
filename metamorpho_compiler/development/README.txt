The system has glibc as a dependency, so it should be installed beforehand
via package manager if possible. It is assumed a user is on Unix/Linux and
has the gcc compiler installed. With these conditions fulfilled, the makefile 
should be able to proceed without problems.

Links for glibc:
https://developer.gnome.org/glib/
https://developer.gnome.org/glib/stable/glib-compiling.html
